🚀 Expense Reimbursement Management System

A scalable and intelligent expense management platform designed to automate and streamline reimbursement workflows for organizations. This system replaces manual, error-prone processes with a dynamic, rule-based approval engine supporting multi-level approvals, conditional logic, and real-time tracking.

🔑 Key Features
Authentication & Role Management
Secure JWT-based login/signup
Automatic company and admin creation on first signup
Role-based access: Admin, Manager, Employee
Expense Submission
Submit expenses with amount, category, date, and description
Multi-currency support with conversion to company currency
Receipt upload with OCR-based auto data extraction
Dynamic Approval Workflow
Multi-level approval system (Manager → Finance → Director)
Configurable approval sequences
Conditional approval rules:
Percentage-based (e.g., 60% approvals)
Specific approver (e.g., CFO override)
Hybrid rules
Real-Time Tracking
Live expense status updates
Approval notifications for managers
Analytics Dashboard
Category-wise and monthly expense insights
Visual charts for better decision-making
Security & Validation
Robust input validation
Role-based authorization
Secure API design
🛠 Tech Stack
Frontend: React
Backend: Node.js + Express
Database: MySQL / PostgreSQL
OCR: Tesseract.js
Charts: Chart.js
